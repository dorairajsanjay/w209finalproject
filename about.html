Better Food Choices was created to help individuals easily find healthier food options. It uses the USDA food database with over 160,000 records on food products to recommend a healthier food alternative.

Navigating through the website

Quick View

 

 

The quick view option can be used to look at product information for a set of already selected products.

 

Search View

The search view can be used to look for products that you are interested in. As the user types out a product a list of relevant products is displayed to help the user find the full text for that product.

Simple View

 

The Simple View allows the user to sort through a list of related products based on a selected category – fat, sugars, proteins etcetera, and visualize this using bar charts.

 

Nutrition Panel

The nutrition panel shows nutrition facts for the selected and recommends products, highlighting the category selected by the user.

Multiple Measures

 

The multiple measures panel provides the user with a single chart that simultaneously compares how two products rank against each other on multiple categories.  

Compare Two

 

The Compare Two panel compares the selected and recommended products using a polygon chart. This allows the user to visualize the differences between products as areas on a two dimensional object.
